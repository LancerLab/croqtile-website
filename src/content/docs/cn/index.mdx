CroqTile 是一种用于 **TileFlow** 编程的 C++ 嵌入式 DSL：你描述带形状的数据、DMA 搬移与并行区域，编译器将其降级为目标代码（例如 CUDA/Cute），并辅以严格的形状与 DMA 安全检查。

如需更深入的参考，请使用下列页面。若是初次接触，请先阅读 **快速开始**（构建与编译器选项），再浏览 **语法设计**，以及与你任务相关的主题（形状、并行、DMA 或动态维度）。
